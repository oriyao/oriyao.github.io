## if statement

if(表达式)语句1 else 语句2

判断表达式的值，如果结果不为0则执行语句1，若为0则执行语句2。

[输入的整数是奇数还是偶数](http://oriyao.oss-cn-hangzhou.aliyuncs.com/website/C/201401/2014010702.c)
    
    #include <stdio.h>
    
    int main(void)
    {
        int num;
    
        printf("输入一个整数：");
        scanf("%d",&num);
    
        if(num%2)
            puts("输入的数是奇数");
        else
            puts("输入的数是偶数"); 
    
        getch();
        return(0);
    }
